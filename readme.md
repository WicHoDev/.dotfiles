# Link files to their designated areas

```
ln -s ~/.DOTFILES/.zshrc ~/.zshrc
ln -s ~/.DOTFILES/.gitconfig ~/.gitconfig
ln -s ~/.DOTFILES/nvim/ ~/.config/
ln -s ~/.DOTFILES/i3/ ~/.config/
```
